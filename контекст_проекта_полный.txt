1. Общая информация о проекте
Проект: Android‑приложение «Анонимные встречи единомышленников».
Платформа: Android Studio, Kotlin, Jetpack Compose.
Данные и авторизация: Firebase Authentication + Firestore.
Геолокация и уведомления: Google Location Services + локальные уведомления.
Карты: Yandex MapKit (MapKitFactory).

Назначение проекта
Создать мобильное приложение, где пользователи могут:
1) создавать и вступать в группы по интересам;
2) общаться в общем чате группы;
3) отправлять приватные сообщения через ключ, который меняется каждый день (UTC);
4) получать уведомление о близости пользователя из общих групп, если оба рядом (50 м);
5) видеть на карте только друзей, которых уже встречали в радиусе 50 м и добавили в друзья;
6) управлять приватностью (ключ, видимость, списки друзей/встреченных, уведомления).

Что было реализовано и доработано по коду в этом чате

2. Архитектура
Проект разделён на:
- data: модели, репозитории, сервисы, преференсы.
- ui: экраны, компоненты, темы.
- services: фоновые сервисы (геолокация, уведомления), MapService.
- di: внедрение зависимостей (Hilt).

Основной паттерн: MVVM
- ViewModel управляет состоянием экрана.
- Repository работает с Firebase и хранением.
- UI в Jetpack Compose подписывается на StateFlow.

3. Модели данных (data/model)
Примеры ключевых моделей:
- Group: id, name, description, category, isPrivate, memberIds, memberLogins, createdAt.
- Message: id, groupId, senderId, senderLogin, text, encryptedText, senderEncryptedText, timestamp, isPrivate, recipientKeyHash, recipientKeyDate, senderKeyHash, senderKeyDate.
- Encounter: userId, login, groupId, groupName, distance, lastSeen.
- Friend: userId, login, addedAt.
- UserLocation: userId, login, latitude, longitude, timestamp.

4. Репозитории и бизнес‑логика

4.1 UserRepository
Назначение:
- регистрация и вход;
- ротация ключей по UTC;
- обновление ника;
- хранение userIds/groupIds;
- доступ к публичным ключам.

Ключевой функционал:
- Генерация ключей раз в сутки (UTC).
- Сохранение ключей в SharedPreferences/DataStore.
- Хэш публичного ключа для фильтрации приватных сообщений.
- При смене ника обновляется user.login и memberLogins в группах.

Фрагмент кода (логика UTC):
"""
private fun currentUtcDate(): String {
    val formatter = SimpleDateFormat("yyyy-MM-dd", Locale.US)
    formatter.timeZone = TimeZone.getTimeZone("UTC")
    return formatter.format(System.currentTimeMillis())
}
"""

4.2 GroupRepository
Назначение:
- загрузка групп пользователя;
- поиск групп;
- создание группы;

4.3 ChatRepository
Назначение:
- общий чат + приватные сообщения.
- комбинирование потоков Firestore:
  - общий чат
  - приватные сообщения для текущего пользователя
  - приватные сообщения, отправленные текущим пользователем

Фрагмент логики объединения:
"""
combine(groupFlow, privateToMeFlow, privateFromMeFlow) { groupMessages, privateToMe, privateFromMe ->
    (groupMessages + privateToMe + privateFromMe)
        .distinctBy { it.id }
        .sortedBy { it.timestamp }
}
"""

4.4 SocialRepository

Логика друзей:

- Удаление — удаляет у обоих.

5. Безопасность и приватность

5.1 Приватные сообщения
- Сообщения в приватном режиме шифруются.
- В Firestore сохраняются:
  - encryptedText для получателя
  - senderEncryptedText для отправителя
- Для получения приватных сообщений используются хэши ключей.
- Сообщения видят только отправитель и получатель.

5.2 Ротация ключей
- Ключ обновляется ежедневно в UTC.
- В профиле отображается текущий ключ и дата.
- Ключ хранится в DataStore.

5.3 Firestore Rules
Добавлены условия:
- читать приватные сообщения может только отправитель или пользователь, чей ключ совпадает с recipientKeyHash.
- удалять записи друзей может любой из двух пользователей.
- join в группу проверяет неизменность некоторых полей.

Фрагмент правил:
"""
match /messages/{messageId} {
  allow read: if isSignedIn() &&
    isMemberOfGroup(resource.data.groupId) && (
      resource.data.isPrivate == false ||
      resource.data.senderId == request.auth.uid ||
      hasKeyHash(resource.data.recipientKeyHash)
    );
  allow create: if isSignedIn() &&
    isMemberOfGroup(request.resource.data.groupId) &&
    request.resource.data.senderId == request.auth.uid;
}
"""

6. Геолокация и уведомления

6.1 LocationService
- Фоновая служба обновляет координаты пользователя.
- Проверяет всех пользователей в user_locations.
- Если расстояние < 50 м и есть общая группа — отправляет уведомление.
- Записывает встречу в encounters.

Алгоритм:
- Получить location.
- Обновить user_locations.
- Проверить всех других пользователей.
- Фильтр: свежий timestamp <= 5 минут.
- Проверить общие группы.
- Если дистанция <= 50 м — уведомление + запись встречи.

6.2 NotificationService

    fun getMessages(groupId: String): Flow<List<Message>> = channelFlow {
        val uid = auth.currentUser?.uid
        if (uid == null) {
            send(emptyList())
            return@channelFlow
        }

        val keyHashes = getRecipientKeyHashes()
        val groupFlow = firebaseService.listenGroupMessages(groupId)
        val privateToMeFlow = if (keyHashes.isNotEmpty()) {
            firebaseService.listenPrivateMessagesByRecipientHashes(groupId, keyHashes)
        } else {
            flowOf(emptyList())
        }
        val privateFromMeFlow = firebaseService.listenPrivateMessagesBySender(groupId, uid)

        combine(groupFlow, privateToMeFlow, privateFromMeFlow) { groupMessages, privateToMe, privateFromMe ->
            (groupMessages + privateToMe + privateFromMe)
                .distinctBy { it.id }
                .sortedBy { it.timestamp }
        }.collect { messages ->
            send(messages)
        }
    }

    private suspend fun getRecipientKeyHashes(): List<String> {
        val dates = userPreferences.getAllKeyDates()
        val hashes = mutableListOf<String>()
        for (date in dates) {
            val pair = userPreferences.getKeyPairForDate(date)
            val publicKey = pair?.first
            if (!publicKey.isNullOrBlank()) {
                hashes.add(encryptionService.hashPublicKey(publicKey))
            }
        }

        if (hashes.isEmpty()) {
            val publicKey = userPreferences.getPublicKey()
            if (!publicKey.isNullOrBlank()) {
                hashes.add(encryptionService.hashPublicKey(publicKey))
            }
        }

        return hashes.distinct()
    }

        val uid = auth.currentUser?.uid ?: throw IllegalStateException("Пользователь не авторизован")

    private val lastNotified = mutableMapOf<String, Long>()
    private val groupNameCache = mutableMapOf<String, String>()

    override fun onCreate() {
        super.onCreate()
        fusedLocationClient = LocationServices.getFusedLocationProviderClient(this)
    }

    override fun onStartCommand(intent: Intent?, flags: Int, startId: Int): Int {
        startForeground(TRACKING_NOTIFICATION_ID, buildTrackingNotification())
        startLocationUpdates()
        return START_STICKY
    }

    private fun startLocationUpdates() {
        val locationRequest = LocationRequest.create().apply {
            priority = Priority.PRIORITY_HIGH_ACCURACY
            interval = 15000L
            fastestInterval = 8000L
        }

        locationCallback = object : LocationCallback() {
            override fun onLocationResult(locationResult: LocationResult) {
                locationResult.lastLocation?.let { location ->
                    serviceScope.launch {
                        updateLocation(location)
                        checkProximity(location)
                    }
                }
            }
        }

        try {
            fusedLocationClient.requestLocationUpdates(
                locationRequest,
                locationCallback,
                Looper.getMainLooper()
            )
        } catch (e: SecurityException) {
            Log.e("LocationService", "Location permission not granted", e)
            stopSelf()
        } catch (e: Exception) {
            Log.e("LocationService", "Error starting location updates", e)
            stopSelf()
        }
    }

    private suspend fun updateLocation(location: Location) {
        val uid = auth.currentUser?.uid ?: return
        val login = userPreferences.getNickname() ?: return
        val groupIds = userPreferences.getEnabledGroups()
        val data = mapOf(
            "latitude" to location.latitude,
            "longitude" to location.longitude,
            "timestamp" to System.currentTimeMillis(),
            "login" to login,
            "groupIds" to groupIds.toList()
        )
        firestore.collection("user_locations")
            .document(uid)
            .set(data)
            .await()
    }

    private suspend fun checkProximity(location: Location) {
        val uid = auth.currentUser?.uid ?: return
        val groupIds = userPreferences.getEnabledGroups().toSet()
        if (groupIds.isEmpty()) return

        val snapshot = firestore.collection("user_locations").get().await()
        val now = System.currentTimeMillis()

        for (doc in snapshot.documents) {
            if (doc.id == uid) continue
            val data = doc.data ?: continue
            val otherLat = data["latitude"] as? Double ?: continue
            val otherLon = data["longitude"] as? Double ?: continue
            val otherTimestamp = data["timestamp"] as? Long ?: continue
            val otherLogin = data["login"] as? String ?: ""
            val otherGroups = (data["groupIds"] as? List<*>)?.filterIsInstance<String>()?.toSet() ?: emptySet()

            if (now - otherTimestamp > 5 * 60 * 1000) continue

            val distance = distanceMeters(location.latitude, location.longitude, otherLat, otherLon)
            if (distance > 50.0) continue

            val commonGroupId = groupIds.intersect(otherGroups).firstOrNull() ?: continue
            val lastTime = lastNotified[doc.id] ?: 0L
            if (now - lastTime < 2 * 60 * 1000) continue

            val groupName = getGroupName(commonGroupId)
            notificationService.showProximityNotification(groupName)
            lastNotified[doc.id] = now

            saveEncounter(uid, doc.id, otherLogin, commonGroupId, groupName, distance)
        }
    }

    private suspend fun getGroupName(groupId: String): String {
        groupNameCache[groupId]?.let { return it }
        return try {
            val doc = firestore.collection("groups").document(groupId).get().await()
            val name = doc.getString("name") ?: groupId
            groupNameCache[groupId] = name
            name
        } catch (e: Exception) {
            groupId
        }
    }

    private suspend fun saveEncounter(
        currentUserId: String,
        otherUserId: String,
        otherLogin: String,
        groupId: String,
        groupName: String,
        distance: Double
    ) {
        val data = mapOf(
            "userId" to otherUserId,
            "login" to otherLogin,
            "groupId" to groupId,
            "groupName" to groupName,
            "distance" to distance,
            "lastSeen" to System.currentTimeMillis()
        )
        firestore.collection("users")
            .document(currentUserId)
            .collection("encounters")
            .document(otherUserId)
            .set(data)
            .await()
    }

    private fun distanceMeters(lat1: Double, lon1: Double, lat2: Double, lon2: Double): Double {
        val results = FloatArray(1)
        Location.distanceBetween(lat1, lon1, lat2, lon2, results)
        return results[0].toDouble()
    }

    private fun buildTrackingNotification(): Notification {
        val channelId = TRACKING_CHANNEL_ID
        val manager = getSystemService(Context.NOTIFICATION_SERVICE) as NotificationManager
        if (Build.VERSION.SDK_INT >= Build.VERSION_CODES.O) {
            val channel = NotificationChannel(
                channelId,
                "Отслеживание местоположения",
                NotificationManager.IMPORTANCE_LOW
            ).apply {
                description = "Фоновое отслеживание местоположения"
            }
            manager.createNotificationChannel(channel)
        }

        return NotificationCompat.Builder(this, channelId)
            .setContentTitle("Отслеживание включено")
            .setContentText("Приложение отслеживает вашу локацию")
            .setSmallIcon(R.drawable.ic_notification)
            .setPriority(NotificationCompat.PRIORITY_LOW)
            .build()
    }

    override fun onBind(intent: Intent?): IBinder? = null

    override fun onDestroy() {
        try {
            if (::locationCallback.isInitialized) {
                fusedLocationClient.removeLocationUpdates(locationCallback)
            }
        } catch (e: Exception) {
            Log.e("LocationService", "Error in onDestroy", e)
        }
        super.onDestroy()
    }

        private const val TRACKING_CHANNEL_ID = "tracking_channel"
        private const val TRACKING_NOTIFICATION_ID = 1001
    }

@OptIn(ExperimentalMaterial3Api::class)
@Composable
fun ChatScreen(

    onNavigateBack: () -> Unit,
    onNavigateToMap: () -> Unit,
    viewModel: ChatViewModel = hiltViewModel()
) {
    val messages by viewModel.messages.collectAsState()
    val groupMembers by viewModel.groupMembers.collectAsState()
    val error by viewModel.error.collectAsState()
    var showMembers by remember { mutableStateOf(false) }
    var messageText by remember { mutableStateOf("") }
    var showPrivateDialog by remember { mutableStateOf(false) }
    var recipientPublicKeyInput by remember { mutableStateOf("") }
    var privateMessageText by remember { mutableStateOf("") }
    val clipboardManager = LocalClipboardManager.current

    fun extractPublicKey(text: String): String? {
        val marker = "Публичный ключ:"
        return if (text.contains(marker)) {
            text.substringAfter(marker).trim()
        } else {
            text.trim().ifBlank { null }
        }
    }

    LaunchedEffect(groupId) {
        viewModel.loadMessages(groupId)
        viewModel.loadGroupMembers(groupId)
    }

    Scaffold(
        topBar = {
            TopAppBar(
                title = { Text("Чат") },
                navigationIcon = {
                    IconButton(onClick = onNavigateBack) {
                        Icon(Icons.Default.ArrowBack, contentDescription = "Назад")
                    }
                },
                actions = {
                    IconButton(onClick = { showMembers = !showMembers }) {
                        Icon(Icons.Filled.Group, contentDescription = "Участники")
                    }
                    IconButton(onClick = onNavigateToMap) {
                        Icon(Icons.Filled.Map, contentDescription = "Карта")
                    }
                }
            )
        }
    ) { paddingValues ->
        ScreenBackground(
            modifier = Modifier.padding(paddingValues)
        ) {
            Column(
                modifier = Modifier.fillMaxSize()
            ) {
                if (messages.isEmpty()) {
                    Box(
                        modifier = Modifier
                            .weight(1f)
                            .fillMaxWidth(),
                        contentAlignment = Alignment.Center
                    ) {
                        Text(
                            text = "Пока нет сообщений",
                            style = MaterialTheme.typography.bodyMedium,
                            color = MaterialTheme.colorScheme.onSurface.copy(alpha = 0.6f)
                        )
                    }
                } else {
                    LazyColumn(
                        modifier = Modifier
                            .weight(1f)
                            .fillMaxWidth(),
                        contentPadding = PaddingValues(horizontal = 16.dp, vertical = 12.dp)
                    ) {
                        items(messages) { message ->
                            MessageItem(message)
                            Spacer(modifier = Modifier.height(10.dp))
                        }
                    }
                }

                Surface(
                    tonalElevation = 2.dp,
                    color = MaterialTheme.colorScheme.surface,
                    modifier = Modifier.fillMaxWidth()
                ) {
                    Row(
                        modifier = Modifier
                            .fillMaxWidth()
                            .padding(12.dp),
                        verticalAlignment = Alignment.CenterVertically
                    ) {
                            OutlinedTextField(
                                value = messageText,
                                onValueChange = { messageText = it },
                                modifier = Modifier.weight(1f),
                                placeholder = { Text("Введите сообщение") }
                            )
                        Spacer(modifier = Modifier.width(8.dp))
                        LongPressButton(
                            onClick = {
                                if (messageText.isNotBlank()) {
                                    viewModel.sendMessage(groupId, messageText)
                                    messageText = ""
                                }
                            },
                            onLongPress = {
                                showPrivateDialog = true
                                if (messageText.isNotBlank()) {
                                    privateMessageText = messageText
                                    messageText = ""
                                }
                            }
                        ) {
                            Icon(Icons.Default.Send, contentDescription = "Отправить")
                        }
                    }
                }
            }

            if (showMembers) {
                AlertDialog(
                    onDismissRequest = { showMembers = false },
                    title = { Text("Участники группы") },
                    text = {
                        Column {
                            groupMembers.forEach { member ->
                                Row(
                                    modifier = Modifier
                                        .fillMaxWidth()
                                        .padding(vertical = 8.dp),
                                    horizontalArrangement = Arrangement.SpaceBetween,
                                    verticalAlignment = Alignment.CenterVertically
                                ) {
                                    Text(member)
                                    IconButton(
                                        onClick = {
                                            viewModel.getPublicKeyByLogin(member) { key ->
                                                if (key != null) {
                                                    clipboardManager.setText(AnnotatedString("Публичный ключ: $key"))
                                                    viewModel.setError("Ключ скопирован в буфер обмена")
                                                } else {
                                                    viewModel.setError("Не удалось получить ключ")
                                                }
                                            }
                                        }
                                    ) {
                                        Icon(Icons.Default.ContentCopy, contentDescription = "Скопировать ключ")
                                    }
                                }
                            }
                        }
                    },
                    confirmButton = {
                        TextButton(onClick = { showMembers = false }) {
                            Text("Закрыть")
                        }
                    }
                )
            }

            if (showPrivateDialog) {
                AlertDialog(
                    onDismissRequest = { showPrivateDialog = false },
                    title = { Text("Приватное сообщение") },
                    text = {
                        Column {
                            OutlinedTextField(
                                value = recipientPublicKeyInput,
                                onValueChange = { recipientPublicKeyInput = it },
                                modifier = Modifier.fillMaxWidth(),
                                label = { Text("Публичный ключ получателя") }
                            )
                            Spacer(modifier = Modifier.height(4.dp))
                            Text(
                                text = "Ключ меняется ежедневно",
                                style = MaterialTheme.typography.labelSmall,
                                color = MaterialTheme.colorScheme.onSurface.copy(alpha = 0.6f)
                            )
                            Spacer(modifier = Modifier.height(8.dp))
                            OutlinedTextField(
                                value = privateMessageText,
                                onValueChange = { privateMessageText = it },
                                modifier = Modifier.fillMaxWidth(),
                                label = { Text("Сообщение") }

                                if (publicKey.isNullOrBlank() || privateMessageText.isBlank()) {

                    modifier = Modifier
                        .align(Alignment.BottomCenter)
                        .padding(16.dp)
                ) {
                    Text(errorMessage)
                }
            }
        }
    }
}

@Composable
fun MessageItem(message: ChatViewModel.UiMessage) {
    val dateFormat = remember { SimpleDateFormat("HH:mm", Locale.getDefault()) }
    val isMine = message.isMine
    val alignment = if (isMine) Alignment.End else Alignment.Start
    val bubbleShape: Shape = if (isMine) {
        RoundedCornerShape(18.dp, 4.dp, 18.dp, 18.dp)
    } else {
        RoundedCornerShape(4.dp, 18.dp, 18.dp, 18.dp)
    }
    val bubbleColor = if (isMine) {
        MaterialTheme.colorScheme.primaryContainer
    } else {
        MaterialTheme.colorScheme.surfaceVariant
    }

    Column(
        modifier = Modifier.fillMaxWidth(),
        horizontalAlignment = alignment
    ) {
        Row(
            horizontalArrangement = Arrangement.spacedBy(8.dp),
            verticalAlignment = Alignment.CenterVertically
        ) {
            val typeLabel = if (message.isPrivate) {
                val hint = message.keyHint?.let { "ключ •$it" } ?: "по ключу"
                "Личное • $hint"
            } else {
                "Общий чат"
            }
            Text(
                text = typeLabel,
                style = MaterialTheme.typography.labelSmall,
                color = MaterialTheme.colorScheme.onSurface.copy(alpha = 0.6f)
            )
            Text(
                text = if (isMine) "Вы" else message.senderLogin,
                style = MaterialTheme.typography.labelMedium
            )
            Text(
                text = dateFormat.format(Date(message.timestamp)),
                style = MaterialTheme.typography.labelSmall
            )
        }
        Spacer(modifier = Modifier.height(4.dp))
        Surface(
            color = bubbleColor,
            shape = bubbleShape
        ) {
            Text(
                text = message.text,
                style = MaterialTheme.typography.bodyMedium,
                modifier = Modifier.padding(horizontal = 12.dp, vertical = 8.dp)
            )
        }
    }
}

fun GroupScreen(
    navController: NavController,
    viewModel: GroupViewModel,
    groupId: String
) {
    val messages by viewModel.messages.collectAsState()
    var messageText by remember { mutableStateOf("") }

    LaunchedEffect(groupId) {
        viewModel.loadMessages(groupId)
    }

    Scaffold(
        topBar = {
            TopAppBar(
                title = { Text("Группа") },
                navigationIcon = {
                    IconButton(onClick = { navController.popBackStack() }) {

                    }
                }
            )
        },
        content = { padding ->
            Column(
                modifier = Modifier
                    .fillMaxSize()
                    .padding(padding)
                    .padding(16.dp)
            ) {
                LazyColumn(
                    modifier = Modifier.weight(1f)
                ) {
                    items(messages) { message ->
                        MessageRow(message)
                    }
                }
                Row(
                    modifier = Modifier.fillMaxWidth(),
                    verticalAlignment = Alignment.CenterVertically
                ) {
                    TextField(
                        value = messageText,
                        onValueChange = { messageText = it },
                        modifier = Modifier.weight(1f),
                        placeholder = { Text("Введите сообщение") }
                    )
                    Spacer(modifier = Modifier.width(8.dp))
                    IconButton(
                        onClick = {
                            if (messageText.isNotBlank()) {
                                viewModel.sendMessage(groupId, messageText)
                                messageText = ""
                            }
                        }
                    ) {
                        Icon(Icons.Filled.Send, contentDescription = "Отправить")
                    }
                }
            }
        }
    )
}

private fun timeUntilNextUtc(): String {
    val tz = TimeZone.getTimeZone("UTC")
    val now = Calendar.getInstance(tz)
    val next = Calendar.getInstance(tz).apply {
        set(Calendar.HOUR_OF_DAY, 0)
        set(Calendar.MINUTE, 0)
        set(Calendar.SECOND, 0)
        set(Calendar.MILLISECOND, 0)
        add(Calendar.DATE, 1)
    }
    val diffMillis = next.timeInMillis - now.timeInMillis
    val hours = diffMillis / (60 * 60 * 1000)
    val minutes = (diffMillis / (60 * 1000)) % 60
    return String.format("%02d:%02d", hours, minutes)
}

    private suspend fun mapToUiMessages(messages: List<Message>): List<UiMessage> {
        val uid = auth.currentUser?.uid
        return messages.mapNotNull { message ->
            val isMine = uid != null && message.senderId == uid
            val text = if (!message.isPrivate) {
                message.text
            } else {
                decryptPrivateMessage(message, isMine)
            }
            val keyHint = if (message.isPrivate) {
                val raw = message.recipientKeyHash ?: message.senderKeyHash
                raw?.takeLast(4)
            } else {
                null
            }

            UiMessage(
                id = message.id,
                senderLogin = if (message.isPrivate) {
                    if (isMine) "Вы" else message.senderLogin
                } else {
                    message.senderLogin
                },
                text = text,
                timestamp = message.timestamp,
                isPrivate = message.isPrivate,
                isMine = isMine,
                keyHint = keyHint
            )
        }
    }

    private suspend fun decryptPrivateMessage(message: Message, isMine: Boolean): String {
        val encrypted = if (isMine) message.senderEncryptedText else message.encryptedText

        if (encrypted.isNullOrBlank()) {
            return "Приватное сообщение"
        }

        val privateKey = findMatchingPrivateKey(message, isMine)
        if (privateKey.isNullOrBlank()) {
            return "Приватное сообщение"
        }

        return try {
            encryptionService.decryptMessage(encrypted, privateKey)
        } catch (e: Exception) {
            "Приватное сообщение"
        }
    }

    private suspend fun findMatchingPrivateKey(message: Message, isMine: Boolean): String? {
        val targetHash = if (isMine) message.senderKeyHash else message.recipientKeyHash
        if (targetHash.isNullOrBlank()) {
            return userPreferences.getPrivateKey()
        }

        val dates = userPreferences.getAllKeyDates()
        for (date in dates) {
            val pair = userPreferences.getKeyPairForDate(date) ?: continue
            val publicKey = pair.first
            val privateKey = pair.second
            if (publicKey.isBlank() || privateKey.isBlank()) continue
            val hash = encryptionService.hashPublicKey(publicKey)
            if (hash == targetHash) {
                return privateKey
            }
        }

        return userPreferences.getPrivateKey()
    }

    data class UiMessage(
        val id: String,
        val senderLogin: String,
        val text: String,
        val timestamp: Long,
        val isPrivate: Boolean,
        val isMine: Boolean,
        val keyHint: String?
    )

package com.example.anonymousmeetup.ui.viewmodels

import android.app.Application
import androidx.lifecycle.AndroidViewModel
import androidx.lifecycle.viewModelScope
import com.example.anonymousmeetup.data.model.UserLocation
import com.example.anonymousmeetup.data.repository.SocialRepository
import com.example.anonymousmeetup.services.MapService
import com.google.firebase.auth.FirebaseAuth
import com.google.firebase.firestore.FirebaseFirestore
import com.google.firebase.firestore.ListenerRegistration
import dagger.hilt.android.lifecycle.HiltViewModel
import kotlinx.coroutines.flow.*
import kotlinx.coroutines.launch
import javax.inject.Inject

@HiltViewModel
class MapViewModel @Inject constructor(
    application: Application,
    private val socialRepository: SocialRepository,
    private val firestore: FirebaseFirestore,
    private val auth: FirebaseAuth,
    val mapService: MapService
) : AndroidViewModel(application) {

    private val _userLocation = MutableStateFlow<UserLocation?>(null)
    val userLocation: StateFlow<UserLocation?> = _userLocation

    private val _friendLocations = MutableStateFlow<List<UserLocation>>(emptyList())
    val friendLocations: StateFlow<List<UserLocation>> = _friendLocations
    val friends = socialRepository.getFriends()
    val encounters = socialRepository.getEncounters()

    fun removeFriend(friendId: String) {
        viewModelScope.launch {
            try {
                socialRepository.removeFriend(friendId)
            } catch (e: Exception) {
                _error.value = "Ошибка удаления из друзей: ${e.message}"
            }
        }
    }
    private val _error = MutableStateFlow<String?>(null)
    val error: StateFlow<String?> = _error

    private val locationListeners = mutableMapOf<String, ListenerRegistration>()
    private val latestLocations = mutableMapOf<String, UserLocation>()
    private var currentFriendIds: Set<String> = emptySet()
    private var currentUserId: String? = null
    private var currentGroupId: String? = null
    private var currentGroupName: String? = null

    init {
        observeFriendsLocations()
    }

    fun setGroupContext(groupId: String?, groupName: String?) {
        currentGroupId = groupId
        currentGroupName = groupName
    }

    private fun observeFriendsLocations() {
        viewModelScope.launch {
            socialRepository.getFriends().collect { friends ->
                val friendIds = friends.map { it.userId }.toSet()
                currentFriendIds = friendIds
                currentUserId = auth.currentUser?.uid

                val desiredIds = if (currentUserId != null) {
                    friendIds + currentUserId!!
                } else {
                    friendIds
                }

                val toRemove = locationListeners.keys - desiredIds
                toRemove.forEach { userId ->
                    locationListeners.remove(userId)?.remove()
                    latestLocations.remove(userId)
                }

                val toAdd = desiredIds - locationListeners.keys
                toAdd.forEach { userId ->
                    val registration = firestore.collection("user_locations")
                        .document(userId)
                        .addSnapshotListener { snapshot, e ->
                            if (e != null) {
                                _error.value = "Ошибка загрузки локаций: ${e.message}"
                                return@addSnapshotListener
                            }
                            if (snapshot == null || !snapshot.exists()) {
                                latestLocations.remove(userId)
                                updateLocations()
                                return@addSnapshotListener
                            }
                            val data = snapshot.data
                            val lat = (data?.get("latitude") as? Number)?.toDouble()
                            val lon = (data?.get("longitude") as? Number)?.toDouble()
                            val timestamp = (data?.get("timestamp") as? Number)?.toLong()
                            if (lat == null || lon == null || timestamp == null) {
                                latestLocations.remove(userId)
                                updateLocations()
                                return@addSnapshotListener
                            }
                            val login = data["login"] as? String ?: userId
                            latestLocations[userId] = UserLocation(
                                userId = userId,
                                login = login,
                                latitude = lat,
                                longitude = lon,
                                timestamp = timestamp
                            )
                            updateLocations()
                        }
                    locationListeners[userId] = registration
                }

                updateLocations()
            }
        }
    }

    private fun updateLocations() {
        val now = System.currentTimeMillis()
        val freshLocations = latestLocations.values.filter { now - it.timestamp <= 5 * 60 * 1000 }
        val uid = currentUserId
        _userLocation.value = freshLocations.firstOrNull { it.userId == uid }
        _friendLocations.value = freshLocations.filter { currentFriendIds.contains(it.userId) }
    }

    fun getGroupContext(): Pair<String?, String?> = Pair(currentGroupId, currentGroupName)

    override fun onCleared() {
        super.onCleared()
        locationListeners.values.forEach { it.remove() }
        locationListeners.clear()
        latestLocations.clear()
        mapService.onStop()
    }
}

===== FILE: D:\Users\mrbor\Desktop\Codex\firestore.rules =====

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    function isSignedIn() {
      return request.auth != null;
    }

    function userData() {
      return get(/databases/$(database)/documents/users/$(request.auth.uid)).data;
    }

    function isMemberOfGroup(groupId) {
      return isSignedIn() &&
        request.auth.uid in get(/databases/$(database)/documents/groups/$(groupId)).data.memberIds;
    }

    function hasKeyHash(hash) {
      return isSignedIn() &&
        userData() != null &&
        hash != null &&
        userData().keyHashes.hasAny([hash]);
    }

    match /users/{userId} {
      allow read: if isSignedIn() && (
        request.auth.uid == userId ||
        (
          userData() != null &&
          resource.data.groupIds.hasAny(userData().groupIds)
        )
      );
      allow create, update: if isSignedIn() && request.auth.uid == userId;
      allow delete: if false;

      match /encounters/{otherUserId} {
        allow read, write: if isSignedIn() && request.auth.uid == userId;
      }

      match /friends/{friendId} {
        allow read: if isSignedIn() && request.auth.uid == userId;
        allow create: if isSignedIn() &&
          (request.auth.uid == userId || request.auth.uid == friendId);
        allow update, delete: if isSignedIn() &&
          (request.auth.uid == userId || request.auth.uid == friendId);
      }

      match /friend_requests/{fromId} {
        allow read: if isSignedIn() && request.auth.uid == userId;
        allow create: if isSignedIn() && request.auth.uid == fromId;
        allow delete: if isSignedIn() && request.auth.uid == userId;
        allow update: if false;
      }
    }

    match /groups/{groupId} {
      allow read: if isSignedIn();
      allow create: if isSignedIn() &&
        request.auth.uid in request.resource.data.memberIds;
      allow delete: if false;
      allow update: if isSignedIn() && (
        request.auth.uid in resource.data.memberIds ||
        (
          request.auth.uid in request.resource.data.memberIds &&
          request.resource.data.name == resource.data.name &&
          request.resource.data.description == resource.data.description &&
          request.resource.data.category == resource.data.category &&
          request.resource.data.isPrivate == resource.data.isPrivate &&
          request.resource.data.createdAt == resource.data.createdAt &&
          request.resource.data.memberIds.hasAll(resource.data.memberIds) &&
          request.resource.data.memberLogins.hasAll(resource.data.memberLogins) &&
          request.resource.data.memberIds.size() == resource.data.memberIds.size() + 1 &&
          request.resource.data.memberLogins.size() == resource.data.memberLogins.size() + 1
        )
      );
    }

    match /messages/{messageId} {
      allow read: if isSignedIn() &&
        isMemberOfGroup(resource.data.groupId) && (
          resource.data.isPrivate == false ||
          resource.data.senderId == request.auth.uid ||
          hasKeyHash(resource.data.recipientKeyHash)
        );
      allow create: if isSignedIn() &&
        isMemberOfGroup(request.resource.data.groupId) &&
        request.resource.data.senderId == request.auth.uid;
      allow update, delete: if false;
    }

    match /user_locations/{userId} {
      allow read: if isSignedIn() && (
        request.auth.uid == userId ||
        exists(/databases/$(database)/documents/users/$(userId)/friends/$(request.auth.uid)) ||
        (
          userData() != null &&
          resource.data.groupIds.hasAny(userData().groupIds)
        )
      );
      allow write: if isSignedIn() && request.auth.uid == userId;
    }
  }
}
